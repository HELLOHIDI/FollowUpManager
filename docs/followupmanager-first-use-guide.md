# FollowUpManager 첫 사용 가이드

처음 사용하는 실무 담당자가 기업을 등록하고, 사업의 집행 기준과 기관 양식을 설정한 뒤 지출을 관리하는 방법을 안내합니다.

이 문서는 아래의 하나의 예시를 끝까지 사용합니다.

| 구분 | 예시 |
| --- | --- |
| 기업 | 주식회사 그랜트랩 |
| 사업 | 2026 기술고도화 사업 |
| 지출 | 외부 강사 자문료 |

> 계정은 별도로 가입하지 않습니다. 관리자에게 받은 이메일과 비밀번호로 로그인하세요.

---

## 1. 로그인하기

### 이 단계에서 하는 일

발급받은 내부 계정으로 FollowUpManager에 로그인합니다. 로그인하면 사업 목록 또는 마지막으로 사용한 사업 대시보드로 이동합니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 이메일 | 관리자에게 받은 로그인 이메일을 입력합니다. | 개인 이메일이 아니라 발급받은 계정을 사용합니다. |
| 비밀번호 | 관리자에게 받은 비밀번호를 입력합니다. | 비밀번호를 모르면 새 계정을 만들지 말고 관리자에게 문의합니다. |

### 따라 하기

1. 로그인 화면에서 이메일과 비밀번호를 입력합니다.
2. **로그인**을 선택합니다.
3. 사업 목록 또는 마지막으로 사용한 사업 대시보드가 보이면 로그인에 성공한 것입니다.

> 화면 캡처 1: 로그인 화면 — 이메일·비밀번호 입력란과 `로그인` 버튼

### 자주 막히는 경우

- 로그인에 실패하면 이메일 철자와 비밀번호를 다시 확인합니다.
- 계정이 없거나 비밀번호를 모르면 관리자에게 계정 발급 또는 초기화를 요청합니다.
- 로그인 후 빈 화면이 보이면 새 사업을 등록하거나, 이미 등록된 사업이 있는지 담당자에게 확인합니다.

---

## 2. 기업 등록하기

### 이 단계에서 하는 일

사업이 속한 기업을 등록합니다. 기업은 하나 이상의 사업을 묶어 관리하는 기준입니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 기업명 | 사업자등록증에 적힌 상호를 그대로 입력합니다. | `주식회사 그랜트랩` |
| 담당자 | 기업을 관리할 담당자를 목록에서 선택합니다. | 실제 업무 담당자를 선택합니다. |
| 기업 형태 | 개인사업자 또는 법인사업자를 선택합니다. | 법인을 선택하면 법인등록번호 입력란이 추가됩니다. |
| 기업 규모 | 실제 기업 규모를 선택합니다. | 아직 모르면 `확인 필요`로 등록한 뒤 수정할 수 있습니다. |
| 사업자등록번호 | 숫자 10자리를 입력합니다. | `123-45-67890`처럼 하이픈을 넣어도 됩니다. |
| 설립일 | 사업자등록증 기준 설립일을 선택합니다. | 미래 날짜는 입력할 수 없습니다. |
| 법인등록번호 | 법인사업자일 때만 숫자 13자리를 입력합니다. | 개인사업자는 비워 둡니다. |

### 따라 하기

1. 사업 목록에서 **기업 추가하기**를 선택합니다.
2. 기업명과 담당자를 입력·선택합니다.
3. 기업 형태와 규모를 선택합니다.
4. 사업자등록번호와 설립일을 입력합니다.
5. 법인사업자라면 법인등록번호를 입력합니다.
6. 필수 항목을 모두 확인한 뒤 **기업 등록**을 선택합니다.

> 화면 캡처 2: 기업 등록 화면 — 기본 정보 입력과 `기업 등록` 버튼

### 자주 막히는 경우

- **기업 등록 버튼이 비활성화됨**: 필수 항목이 비어 있거나 번호·날짜 형식이 맞지 않습니다.
- **사업자등록번호 중복 안내**: 이미 등록된 기업입니다. 기존 기업을 찾아 사용하거나 담당자에게 확인합니다.
- **법인등록번호 입력란이 보이지 않음**: 기업 형태가 개인사업자로 선택된 상태입니다.

---

## 3. 사업 등록하기

### 이 단계에서 하는 일

등록한 기업에 사업을 연결하고, 협약·예산·담당자 정보를 입력합니다. 사업을 등록하면 해당 사업의 대시보드와 집행 기준 설정 화면을 사용할 수 있습니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 사업명 | 내부에서 구분할 사업명을 입력합니다. | `2026 기술고도화 사업` |
| 주관기관 | 협약을 체결한 기관명을 입력합니다. | `한국산업기술진흥원` |
| 과제명·과제번호 | 협약서에 적힌 과제명과 번호를 입력합니다. | 과제번호는 선택 항목입니다. |
| 협약 기간 | 실제 협약 시작일과 종료일을 선택합니다. | 종료일은 시작일보다 빠를 수 없습니다. |
| 사업 담당자 | 담당자명과 이메일 또는 연락처 중 하나 이상을 입력합니다. | 연락 가능한 정보 하나는 반드시 필요합니다. |
| 총사업비 | 사업 전체 예산을 원 단위로 입력합니다. | `100,000,000` |
| 재원 비율 | 정부지원금·현금·현물의 비율을 입력합니다. | 세 비율의 합계가 100%여야 합니다. |
| 참고 파일 | 협약서, 기관 안내문, 제출 양식 등을 첨부합니다. | 파일당 최대 20MB입니다. |

### 따라 하기

1. 기업 카드에서 **사업 등록**을 선택합니다.
2. 사업명, 주관기관, 과제명과 과제번호를 입력합니다.
3. 협약 시작일·종료일과 사업 담당자 정보를 입력합니다.
4. 총사업비와 재원 비율을 입력하고, 화면에 계산된 금액을 확인합니다.
5. 협약서나 기관 안내문이 있다면 참고 파일로 첨부합니다.
6. **사업 등록**을 선택합니다.

> 화면 캡처 3: 사업 등록 화면 — 협약 정보, 예산 구성, 참고 파일 첨부 영역

### 자주 막히는 경우

- **등록 버튼이 비활성화됨**: 필수 항목이 빠졌거나 재원 비율 합계가 100%가 아닙니다.
- **과제번호 중복 안내**: 같은 기업에 같은 과제번호가 등록되어 있는지 확인합니다.
- **파일을 첨부하지 못함**: 허용된 파일 형식인지, 파일 크기가 20MB 이하인지 확인합니다.

---

## 4. 집행 기준 서류를 추출하고 확정하기

### 이 단계에서 하는 일

협약서나 집행 기준 PDF에서 비목과 필요한 증빙서류를 추출한 초안을 검토하고, 이 사업에서 사용할 집행 기준으로 확정합니다. 확정한 기준은 이후 지출 등록과 증빙 안내에 반영됩니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 기준 PDF | 협약서·사업비 집행 기준처럼 비목과 증빙 규정이 담긴 PDF를 올립니다. | PDF만 자동 추출 대상입니다. |
| 추출 비목 | 사업에 실제 적용할 비목명과 하위항목을 확인·수정합니다. | 기관 문서의 표현과 다르면 이해하기 쉬운 이름으로 조정합니다. |
| 증빙서류 | 비목별로 필요한 증빙서류와 필수 여부를 확인·수정합니다. | 계약서, 견적서, 세금계산서, 결과보고서 등 |
| 필수 여부 | 각 증빙을 `필수`, `조건부`, `선택`으로 지정합니다. | 조건부는 특정 거래에만 필요할 때 사용합니다. |

### 따라 하기

1. 사업 등록 후 열리는 사업 설정 화면에서 집행 기준 PDF를 업로드합니다.
2. 추출이 끝나면 비목별 집행 증빙서류 초안을 확인합니다.
3. 비목명, 하위항목, 증빙서류 이름이 실제 기관 기준과 맞는지 검토합니다.
4. 필요한 비목·증빙서류를 추가하거나, 불필요한 항목을 삭제합니다.
5. 각 증빙서류의 필수 여부를 지정합니다.
6. 검토가 끝나면 **정책 확정**을 선택합니다.

> 화면 캡처 4: 집행 기준 설정 화면 — PDF 업로드와 비목별 증빙서류 초안 검토 표

### 자주 막히는 경우

- **PDF에서 추출할 텍스트가 부족함**: 스캔 이미지형 PDF일 수 있습니다. 기본 비목으로 시작하거나 문서의 텍스트를 붙여넣어 다시 시도합니다.
- **추출 결과가 실제 기준과 다름**: 초안은 자동 제안일 뿐입니다. 기관 문서를 기준으로 비목과 증빙서류를 수정한 뒤 확정합니다.
- **이미 지출을 등록했는데 기준을 바꾸려 함**: 등록된 지출이 있으면 비목 정책을 교체할 수 없습니다. 현재 기준을 유지하고 필요한 지출 정보를 보완합니다.

---

## 5. 기관 양식을 증빙서류에 연결하기

### 이 단계에서 하는 일

기관에서 제공한 제출 양식을 업로드하고, 앞에서 확정한 증빙서류 유형에 직접 연결합니다. 연결한 양식은 지출 상세 화면에서 다운로드해 사용할 수 있습니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 기관 양식 파일 | 기관에서 배포한 제출 양식을 업로드합니다. | HWP, DOCX, XLSX, PDF 등을 올릴 수 있습니다. |
| 증빙서류 유형 | 양식이 필요한 증빙서류 행을 선택합니다. | `결과보고서`, `집행요청서` 등 |
| 연결 | 업로드한 양식을 해당 증빙서류 유형으로 끌어 놓거나 연결 동작을 사용합니다. | 하나의 증빙 유형에 여러 양식을 연결할 수 있습니다. |
| 사용자 정의 증빙서류 | 목록에 없는 기관 서류 유형을 추가합니다. | 기관 고유 서식이 있을 때만 사용합니다. |

### 따라 하기

1. 집행 기준을 확정한 뒤 **기관 양식 연결**을 선택합니다.
2. **파일 추가**로 기관 양식을 업로드합니다.
3. 오른쪽 증빙서류 목록에서 양식을 사용할 증빙서류 유형을 찾습니다.
4. 업로드한 양식을 해당 증빙서류 유형에 연결합니다.
5. 목록에 없는 기관 고유 서식은 **증빙서류 추가**로 만든 뒤 양식을 연결합니다.
6. 연결 상태를 확인하고 **저장**을 선택합니다.

> 화면 캡처 5: 기관 양식 연결 화면 — 왼쪽 양식 파일 목록과 오른쪽 증빙서류 유형 목록

### 자주 막히는 경우

- **양식이 자동으로 연결되지 않음**: 기관 양식은 자동 매칭하지 않습니다. 필요한 증빙서류 유형에 직접 연결합니다.
- **연결할 증빙서류가 없음**: 먼저 집행 기준에서 비목·증빙서류를 확정했는지 확인합니다.
- **기관 양식이 아직 없음**: 이 단계는 건너뛸 수 있습니다. 나중에 사업 관리에서 같은 화면으로 돌아와 업로드·연결할 수 있습니다.
- **저장하지 않고 나가려 함**: 변경사항이 있으면 저장한 뒤 이동합니다.

---

## 6. 사업 대시보드 확인하기

### 이 단계에서 하는 일

사업의 예산 현황과 지출 진행 상태를 한 화면에서 확인합니다. 대시보드는 지출을 등록하기 전 사업의 기본 상태를 점검하고, 등록 후에는 진행 상황을 보는 곳입니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 총예산 | 사업 등록 때 입력한 총사업비입니다. | 사업 전체 예산 기준입니다. |
| 소진액 | `집행 완료` 단계 지출 금액의 합계입니다. | 집행 요청 단계 금액은 포함되지 않습니다. |
| 잔액 | 총예산에서 소진액을 뺀 금액입니다. | 음수이면 데이터 확인이 필요합니다. |
| 예산소진율 | 협약 기간 대비 예산이 얼마나 집행되었는지 보여줍니다. | 높은 비율은 우선 점검합니다. |
| 비목별 지출 | 비목별 지출 건수와 합계를 확인합니다. | 비목을 펼치면 개별 지출을 볼 수 있습니다. |
| 지출 칸반 | 지출의 현재 단계를 확인합니다. | 사업비 등록부터 집행 완료까지 5단계입니다. |

### 따라 하기

1. 사업 대시보드의 **개요**에서 총예산, 소진액, 잔액을 확인합니다.
2. **비목별 지출**에서 비목을 펼쳐 지출 제목·금액·단계를 확인합니다.
3. **지출 칸반**에서 지출이 어느 단계에 있는지 확인합니다.
4. 새 지출을 만들려면 오른쪽 상단의 **+ 지출**을 선택합니다.

> 화면 캡처 6: 사업 대시보드 — 개요 KPI, 비목별 지출, 지출 칸반

### 자주 막히는 경우

- **소진액이 예상과 다름**: `집행 완료` 지출만 소진액에 반영됩니다.
- **비목별 지출이 비어 있음**: 해당 비목으로 지출을 아직 등록하지 않은 상태입니다.
- **대시보드에 사업이 보이지 않음**: 올바른 사업을 선택했는지, 사업 등록이 완료됐는지 확인합니다.

---

## 7. 지출을 등록하고 단계별로 관리하기

### 이 단계에서 하는 일

지출 1건을 등록하고, 상세 화면에서 필요한 정보를 누적 입력하면서 `사업비 등록 → 사전 승인 → 집행 수행 → 집행 요청 → 집행 완료` 순서로 관리합니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 지출 제목 | 나중에 쉽게 찾을 수 있는 업무명으로 입력합니다. | `외부 강사 자문료` |
| 비목·하위비목 | 확정한 집행 기준에 맞는 비목을 선택합니다. | 외주용역비 → 자문료 |
| 재원 구분 | 정부지원금·현금·현물 중 해당 재원을 선택합니다. | 정부지원금 |
| 금액 | 원 단위 금액을 입력합니다. | `3,000,000` |
| 지출 예정일·거래처 | 알 수 있는 범위에서 입력합니다. | 예정일과 거래처는 이후에도 수정할 수 있습니다. |
| 단계별 정보 | 각 단계에서 실제 진행 현황과 관련 정보를 추가합니다. | 아래 표를 참고합니다. |

### 따라 하기

1. 대시보드에서 **+ 지출**을 선택합니다.
2. 지출 제목, 비목, 재원 구분, 금액을 입력합니다.
3. 지출 예정일과 메모를 필요한 만큼 입력한 뒤 **등록**을 선택합니다.
4. 생성된 지출 카드 또는 비목별 지출 목록에서 해당 지출을 선택합니다.
5. 상세 화면에서 현재 단계의 정보를 저장합니다.
6. 상단의 단계 이동 동작으로 바로 다음 단계로 이동합니다.
7. 집행이 끝나면 `집행 완료` 단계에서 최종 정보를 확인합니다.

> 화면 캡처 7: 지출 빠른 등록 화면 — 제목·비목·재원 구분·금액 입력

> 화면 캡처 8: 지출 상세 화면 — 5단계 진행 표시와 현재 단계 입력 영역

#### 단계는 이렇게 이동합니다

| 단계 | 이때 입력·확인할 내용 | 대표 증빙 또는 주의사항 |
| --- | --- | --- |
| 사업비 등록 | 비목, 금액, 예정일, 거래처, 기본 메모 | 이후 단계의 기준이 되므로 비목과 금액을 먼저 확인합니다. |
| 사전 승인 | 승인 필요 여부·상태, 승인번호 또는 링크, 비교견적 준비 여부 | 승인 관련 파일이나 비교견적이 있으면 함께 관리합니다. |
| 집행 수행 | 수행 시작일·종료 예정일, 계약서 또는 발주서, 수행 내용·상태 | 실제 수행 상황을 팀이 알 수 있게 기록합니다. |
| 집행 요청 | 최종 집행 금액, 요청일·상태, 지급 관련 정보 | 영수증·세금계산서·결과보고서 등 증빙을 점검합니다. |
| 집행 완료 | 최종 정산 내용과 완료 시점 | 이 단계의 금액만 대시보드 소진액에 반영됩니다. |

### 자주 막히는 경우

- **다음 단계로 바로 갈 수 없음**: 단계는 한 번에 하나씩 앞으로만 이동합니다. 현재 단계의 저장 상태를 먼저 확인합니다.
- **누락 안내가 표시됨**: 안내는 보완해야 할 정보를 알려주는 것입니다. 데이터 오류가 아니라면 안내를 확인한 뒤 계속 진행할 수 있습니다.
- **이전 단계 정보를 수정해야 함**: 지출 상세에서 이전 단계의 입력 영역을 열어 수정할 수 있습니다.
- **집행 완료로 이동했는데 소진액이 반영되지 않음**: 새로고침 후에도 같다면 지출 금액과 현재 단계를 확인합니다.

## 8. 지출 증빙 파일 관리하기

### 이 단계에서 하는 일

지출 상세 화면에서 단계별 증빙서류를 업로드하고, 필요한 경우 열어 보거나 삭제합니다. 앞에서 확정한 집행 기준과 연결한 기관 양식은 이 단계의 증빙 안내에 반영됩니다.

### 입력·확인할 항목

| 항목 | 어떻게 입력·확인하나요? | 예시 또는 주의사항 |
| --- | --- | --- |
| 증빙서류 유형 | 현재 단계에 필요한 증빙서류 행을 확인합니다. | `세금계산서`, `결과보고서`, `검수조서` |
| 필수 여부 | `필수`, `조건부`, `선택` 상태를 확인합니다. | 필수 누락은 특히 집행 요청 전에 점검합니다. |
| 파일 선택 | 해당 증빙서류 행의 **파일 선택**으로 파일을 올립니다. | 하나의 증빙 유형에 여러 파일을 추가할 수 있습니다. |
| 기관 양식 | 연결된 기관 양식이 있으면 내려받아 작성합니다. | 작성한 파일을 같은 증빙서류 유형에 올립니다. |
| 업로드 파일 | 파일명과 업로드 상태를 확인합니다. | **열기**로 내용을 확인하고, 잘못 올린 파일은 삭제합니다. |

### 따라 하기

1. 지출 상세 화면에서 현재 단계의 증빙 파일 영역을 엽니다.
2. 필요한 증빙서류 유형을 찾고, 필수 여부를 확인합니다.
3. 기관 양식이 연결되어 있다면 양식을 내려받아 작성합니다.
4. **파일 선택**으로 작성·수집한 증빙 파일을 업로드합니다.
5. 업로드가 끝나면 파일명 오른쪽의 **열기**로 내용이 맞는지 확인합니다.
6. 잘못 올린 파일은 **삭제**하고 올바른 파일을 다시 업로드합니다.

> 화면 캡처 9: 지출 상세의 증빙 파일 영역 — 파일 선택, 기관 양식, 업로드 파일의 `열기`·`삭제`

### 자주 막히는 경우

- **어떤 파일을 올려야 할지 모름**: 증빙서류 행의 이름과 필수 여부를 확인하고, 기관 양식이 있다면 내려받아 작성합니다.
- **같은 파일을 여러 번 올림**: 중복 가능성 안내가 보이면 기존 파일과 비교한 뒤 필요한 파일만 남깁니다.
- **파일이 열리지 않음**: 업로드가 끝났는지 확인하고 다시 시도합니다. 계속 실패하면 파일 형식과 크기를 확인합니다.
- **필수 증빙이 없어도 다음 단계로 이동됨**: 이동이 허용되더라도 집행 요청 전에 필수 증빙을 반드시 다시 점검합니다.
